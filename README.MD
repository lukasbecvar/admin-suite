## Description
Simple & user-friendly solution for monitoring and managing Linux servers, primarily designed for single server instances debian-based systems.

## Documentation
Documentation is available at [documentation](DOC.MD).

## Requirements
* NPM package manager and assets build tool for encore build
* Composer for managing PHP dependencies
* PHP 8.4
* MySQL 8 or higher
* MySQL-PDO extension (db driver)
* PHP-OpenSSL extension
* PHP-Intl extension
* PHP-Curl extension
* PHP-GD extension

## Development environment
The complete stack for development is in Docker Compose, including PHP, Node, and Composer. You just need to run the docker-start script to launch it.
